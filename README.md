# E-Learning SS DELTA

Repository ini berisi modul pembelajaran interaktif DELTA untuk Suggestion System.

## Isi Repository

- `index.html` - halaman masuk untuk membuka modul, berguna saat GitHub Pages diaktifkan.
- `modul_delta_interaktif.html` - modul DELTA interaktif utama.
- `SUGESSTION SYSTEM with DELTA.pdf` - referensi PDF yang ditampilkan langsung di dalam modul.

## Cara Membuka

1. Buka `index.html` atau `modul_delta_interaktif.html` di browser.
2. Pastikan file PDF tetap berada di folder yang sama dengan file HTML.
3. Jika menggunakan GitHub Pages, gunakan root repository sebagai sumber publikasi.

## Catatan

Preview PDF di dalam HTML bergantung pada dukungan browser. Jika preview tidak tampil, gunakan tombol **Buka PDF** di dalam modul.
